---
name: 요청하지 않은 요소 추가 금지
description: 사용자가 요청하지 않은 텍스트·UI 요소를 임의로 추가하지 말 것
type: feedback
---

수정 시 사용자가 명시적으로 요청하지 않은 요소(서브타이틀, 설명 텍스트 등)를 임의로 추가하지 않는다.

**Why:** 디자이너가 "서브타이틀 제거"를 요청했는데 다른 서브타이틀을 새로 만들어 넣어서 재수정이 필요했음.

**How to apply:** 제거 요청 시에는 제거만 하고, 새로운 요소를 대체로 넣고 싶다면 반드시 먼저 사용자에게 확인을 받는다.
