# PRD: 설문조사 페이지 리디자인

## 개요
잡코리아 모바일 설문조사 페이지를 JAMS 2.0 디자인 시스템에 맞게 리디자인한다.

## 개선이 필요한 이유

### 1. 시각적 위계 부족
현재 페이지는 제목, 날짜, 본문, 설문 정보가 동일한 톤으로 나열되어 있어 사용자가 핵심 정보를 빠르게 파악하기 어렵다. 제목 크기가 작고, 참여 보상(신세계 상품권)이 본문에 묻혀 있어 설문 참여 동기를 유발하지 못한다.

### 2. CTA(참여 유도) 비효율
"설문시작!" 버튼이 일러스트레이션 형태로 되어 있어 탭 가능한 버튼인지 직관적이지 않다. 스크롤 시 CTA가 화면에서 사라져 참여 전환율을 떨어뜨린다.

### 3. 디자인 시스템 미적용
헤더, 버튼, 타이포그래피, 컬러 등이 JAMS 2.0 컴포넌트 규격을 따르지 않아 잡코리아 앱 내 다른 화면과 시각적 일관성이 떨어진다.

### 4. 불필요한 정보 과다
푸터 영역(로그아웃, 고객센터, 이용약관 등)이 설문 참여라는 목적과 무관하게 화면을 차지하고 있어 집중도를 낮춘다.

## 주요 변경사항

| 항목 | As-is | To-be | 개선 효과 |
|------|-------|-------|----------|
| 헤더 | 파란 배경 + 뒤로가기 | 흰 배경 + X 닫기 | 콘텐츠 집중도 향상 |
| 제목 | 본문 크기, 1줄 | 26px Bold + 이모지 | 가독성·친근감 향상 |
| 날짜 | 제목 아래, 회색 | 제목 위, Primary 컬러 | 기간 인지 즉시 가능 |
| 보상 안내 | 텍스트만 | 커피 이미지 + 볼드 강조 | 참여 동기 유발 강화 |
| 정보 영역 | 컬러 뱃지 나열 | 카드형 정리 | 정보 스캔 용이 |
| CTA | 일러스트 버튼 | 하단 고정 버튼 | 전환율 향상 기대 |
| 푸터 | 전체 서비스 링크 | 제거 | 불필요 요소 제거 |
